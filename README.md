## Configuration
1. Clone repository to your computer.
2. Create build directory ```mkdir build```
3. Go to the build ```cd build```
4. Create CMake files ```cmake ..```
5. Build ```make```
6. Run ```./src/CaveStory```

## Directories
- data - assets from the original game,
- lib - libraries that keep the game running,
- src - main function,
- tst - Google Tests.
